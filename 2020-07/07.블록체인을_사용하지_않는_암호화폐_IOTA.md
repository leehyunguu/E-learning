# 블록체인을 사용하지 않는 암호화폐 IOTA

## IOTA

- 사물인터넷 환경에 최적화된 암호화폐

### Tangle 기반의 IOTA(2015)

- Directed Acyclic Graph
- 여러 방향으로 뻗어나가는 구조
- IOT 환경과 유사
- 초소형, 경량화되어 저전력으로 구동

### Tangle의 주요과제

- 전체 장부를 만드는 사람 중 1/3 이상만 담합을 하면 장부의 내용 조작 가능
- 중구난방식 증가로 인해 옳은 방향을 이끌어주는 Coordinator(중앙)이 필요
- 네트워크 문제점

### One Time Signature

- 양자 컴퓨터에 강한 내성을 가지게 됨
