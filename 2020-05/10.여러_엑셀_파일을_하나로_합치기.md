# 여러 엑셀 파일을 하나로 합치기

## 엑셀 파일을 읽는 2가지 방법

### 엑셀 프로그램 없이 파이썬 라이브러리만으로 처리

- pd.read_excel('엑셀파일경로')
- GUI 엑셀 프로그램을 열 필요없이, 파이썬 단에서 처리
- 단순히 데이터만 읽어서 합치는 경우에 보다 빠른 처리

### 엑셀 프로그램을 통한 처리

- xlwings 라이브러리 활용
- GUI 엑셀 프로그램을 열어야 하기에 파일이 많을 경우, 비효율적일 수 있음.
- 엑셀 기능을 활용한 자동화
- 엑셀파일 포맷 rows 개수 제한

> 65,535(xls) / 2,048,576(xlsx)
